# Ketu

An AI-aware HTTP request router and load balancer, built in Rust.

Ketu sits between clients and multiple LLM backends, intelligently routing 
traffic based on semantic complexity, enforcing rate limits, tracking latency 
and cost, and coordinating across distributed instances.
